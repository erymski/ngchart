How to use it
=============
1. Extract assembly from the downloaded archive.
2. Add reference from the assembly to your project.
3. Add "using NGChart;" to CS file.
4. Use samples from Example page (http://code.google.com/p/ngchart/wiki/NGChartSamples) as guide.
5. Or check code from unit tests (http://ngchart.googlecode.com/svn/trunk/NGChartTests/)
